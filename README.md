# Geo Data Dashboard

## Overview
React-based dashboard to visualize spatial and tabular geo data.

## Tech Stack
- React (Vite)
- MUI DataGrid
- React Leaflet

## Architecture Decisions
- Local state for simplicity
- Custom hook for data logic
- Virtualized table for performance

## Features
- Paginated, sortable data table
- Client-side filtering
- Interactive map with synced selection

## Performance
- Handles 5k+ rows using virtualization
- Memoized filtering and sorting
